# Rapid Prototyping

A collection of HTML prototypes and design assets for rapid iteration and stakeholder review.

## Structure

```
arisdesign/
├── Assets/           # Shared images, icons, and design tokens
│   ├── icons/
│   ├── images/
│   └── design-tokens/
└── [design-topic]/   # Each design topic gets its own folder
    └── *.html        # HTML prototype files
```

## Live Preview

All HTML files are viewable via GitHub Pages at:
```
https://arisdesign.github.io/early-concepts/
```

## Usage

Each design topic folder contains self-contained HTML prototypes that can be opened directly in a browser or shared via GitHub Pages links.
